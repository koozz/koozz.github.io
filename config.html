NAME

koozz-config — editor and dotfiles configuration

SYNOPSIS

nvim [file ...]

DESCRIPTION

Primary editor is Neovim. Configuration is always in flux — plugins come and go, keybindings shift, and the colorscheme changes with the seasons.

Terminal emulator is Ghostty, a GPU-accelerated terminal that is fast, feature-rich, and native on both macOS and Linux.

FILES
~/.config/nvim/init.lua

Main Neovim configuration file. The latest version is hosted at codeberg.org/koozz/dotfiles.

~/.config/ghostty/config

Ghostty terminal configuration.

ENVIRONMENT
  • EDITOR=nvim
  • SHELL=/opt/homebrew/bin/fish
  • TERM=ghostty
